ROOT CAUSE ANALYSIS
===================

Main Issues:
1. Log4j Error Root Cause:
   - Strimzi sets KAFKA_LOG4J_OPTS=-Dlog4j.configuration=file:/opt/kafka/custom-config/log4j.properties by default
   - This path needs to exist OR be overridden
   - Fix: Either copy log4j.properties to custom-config OR change KAFKA_LOG4J_OPTS

2. Bootstrap Servers Empty:
   - Strimzi generates bootstrap.servers from spec.bootstrapServers
   - May need explicit bootstrap.servers in spec.config as well
   - Full DNS name sometimes needed: my-cluster-kafka-bootstrap.lakehouse-ingest.svc.cluster.local:9092

3. Image Pull Caching:
   - Using :latest tag causes caching issues
   - Solution: Use explicit version tags and imagePullPolicy: Always
